# weather-findings---pandas--
# Weather Findings using Pandas

A data analysis project focused on exploring, cleaning, and analyzing weather dataset patterns using Python and the Pandas library in Google Colab.

---

## 🚀 Features

*   **Data Cleaning:** Handling missing values, correcting data types, and preparing weather logs for analysis.
*   **Weather Metrics Analysis:** Analyzing temperature variations, wind speed, visibility, and weather conditions (like Clear, Foggy, Snow, etc.).
*   **Query-Based Filtering:** Finding specific weather instances (e.g., times when it was exactly clear or when visibility was above a certain threshold).

---

## 🛠️ Tech Stack

*   **Language:** Python 3.x
*   **Environment:** Google Colab / Jupyter Notebook (`weather.ipynb`)
*   **Libraries:** Pandas, NumPy

---

## 📊 Dataset Description

The project utilizes a dataset named `Weather Dataset.csv`, which contains time-series data for weather conditions at a particular location with the following parameters:
*   Date/Time
*   Temperature (C)
*   Dew Point Temperature (C)
*   Relative Humidity (%)
*   Wind Speed (km/h)
*   Visibility (km)
*   Pressure (kPa)
*   Weather Condition

---

## 💻 Getting Started & How to Use

Since this project is optimized for **Google Colab**, follow these steps to run the analysis:

### 1. Setup the Notebook
* Upload `weather.ipynb` and `Weather Dataset.csv` to your Google Drive or open the notebook directly in [Google Colab](https://colab.research.google.com/).

### 2. Upload the Dataset to Colab
Before running the code, make sure to upload the `Weather Dataset.csv` file into your Colab runtime session (click the folder icon on the left sidebar and drag the CSV file there).

### 3. Run the Analysis
* Go to the top menu in Colab and click **Runtime > Run all** (or press `Ctrl + F9`).
* Walk through the cells to see data insights, filtered views, and statistics.
